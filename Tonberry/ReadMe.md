# 虚拟冬贝利玩偶

　　本模型为基于「最终幻想 14」/「FINAL FANTASY XIV」中的生物「冬贝利」的形象制作的**免费** Live2D 模型。

　　作者：Achrom-4o

　　配布地：bilibili @ 观月盐Re｜Github @ Hekatech

## 按键说明

### 屏幕按键

1. 帽子显示 / 隐藏
2. 菜刀拿起 / 放下
3. 提灯拿起 / 放下
4. 灯光开 / 关

### 键盘按键

f = 菜刀高光
LShift + N6 = 水滴
RShift = 问号 (sb VTS 不让设问号快捷键我服了)
N1 = 生气
